This project analyzes a Superstore sales dataset using Python.  
The goal of the project was to understand the sales performance, profit distribution, and monthly trends from the dataset.

## My Dataset
The dataset contains around 9,950 records with information such as:
Order ID, Order Date, Ship Date, Ship Mode, Segment, Region, Category, Sales, Quantity, Discount, and Profit.

## What I did in this project
- Loaded the dataset using Pandas
- Removed duplicate and unnecessary columns
- Converted date columns into proper datetime format
- Created new features such as Month and Year
- Performed exploratory data analysis
- Generated visualizations to understand sales and profit trends

## My Visualizations
The following charts were generated:

- Category-wise Sales
- Region-wise Profit
- Monthly Sales Trend

These charts help understand which categories generate more sales, which regions are more profitable, and how sales change over time.
